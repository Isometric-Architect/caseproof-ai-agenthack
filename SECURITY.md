# Security Policy

CaseProof AI is a public-safe hackathon prototype. Do not place credentials, real customer records, tenant secrets, private verifier internals, private negative-control banks, or hidden thresholds in this repository.

If you find sensitive material in this repository, remove it before public upload and rotate any affected credential outside this repository.

This prototype does not claim production security readiness, legal compliance, financial authorization, or enterprise approval authority.
